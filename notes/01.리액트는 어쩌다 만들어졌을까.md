Virtual DOM
- 가상의 DOM 
- 브라우저에 보여주는 DOM이 아닌 메모리에 가상으로 존재하는 DOM임. 자바스크립트 객체.여서 속도가 빠름
- 상태 업데이트 -> 업데이트가 필요한 곳 UI를 Virtual DOM을 통해서 렌더링 -> 실제 DOM이랑 비교 -> 차이 있는 곳 DOM 패치
